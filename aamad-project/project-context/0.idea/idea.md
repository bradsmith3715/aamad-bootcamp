# Project Name: Origin AI

## What it does: An AI-powered project scoping tool for solo developers that conducts discovery conversations, generates planning documents, and produces actionable build plans.

## Who it's for: Solo developers who want to quickly scope and plan new project ideas.

## Problem it solves: Reduces the time and effort needed to go from a rough idea to a structured, actionable development plan.

## Key features:

- Conversational discovery to clarify project requirements
- Auto-generates MRD/PRD-style planning docs
- Tech stack recommendations
- Similar GitHub repo suggestions
- End-to-end build plan
